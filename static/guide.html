DNS-over-TLS Configuration

Android natively supports DNS-over-TLS since Android 9 (Pie). Here's how to set it up:

  1. Open Settings

    On your Android device, open the Settings app.

  2. Go to Network & Internet

    Navigate to "Network & Internet" or "Connections" (may vary depending on your device).

  3. Select "Private DNS"

    Look for "Private DNS" option. On some devices, you might need to tap on "More Connection Settings" or "Advanced" to find it.

  4. Enter your WebShield hostname

    Select "Private DNS provider hostname" or "Enter provider hostname" and enter your WebShield DoT address:

    {{.ConfigId}}.{{.Host}}

  5. Save and verify

    Tap "Save" or "Apply" and make sure the Private DNS setting shows as "Connected" or "Active".

Note: If you cannot find the Private DNS option, your device might be running an older version of Android that doesn't support DNS-over-TLS. In that case, maybe try configuring browsers to use DoH instead.

Browser DNS-over-HTTPS Configuration

  1. Open Chrome Settings

    Click the three dots (⋮) in the top right corner and select "Settings".

  2. Navigate to Privacy and Security

    In the left menu, click on "Privacy and security".

  3. Select Security

    Click on "Security" in the submenu.

  4. Configure Secure DNS

    Scroll down to find "Use secure DNS" and make sure it's enabled.

  5. Choose Custom provider

    Select "With Custom" and enter your WebShield DoH URL:

    https://{{.Host}}/doh/{{.ConfigId}}

  1. Open Firefox Settings

    Click the hamburger menu (≡) in the top right and select "Settings" or "Preferences".

  2. Navigate to Network Settings

    Scroll down to the bottom and click on "Settings" under the "Network Settings" section.

  3. Enable DNS over HTTPS

    Check the box for "Enable DNS over HTTPS".

  4. Use custom provider

    Select "Custom" and enter your WebShield DoH URL:

    https://{{.Host}}/doh/{{.ConfigId}}

  5. Save and close

    Click "OK" to save the settings.

  1. Open Edge Settings

    Click the three dots (…) in the top right corner and select "Settings".

  2. Navigate to Privacy, search, and services

    Select "Privacy, search, and services" from the left menu.

  3. Scroll to Security

    Scroll down to find the "Security" section.

  4. Configure Secure DNS

    Toggle on "Use secure DNS to specify how to lookup the network address for websites".

  5. Choose custom provider

    Select "Choose a service provider" and click "Enter custom provider".

    Enter your WebShield DoH URL:

    https://{{.Host}}/doh/{{.ConfigId}}

iOS and iPadOS Configuration

Create a custom configuration profile to enable encrypted DNS on your iOS device. We have prefilled most of the details. If you want to change anything, you can do so.


  1. Profile Settings

  2. DNS Settings

  3. Configuration Profile Code

    If you want to edit anything you can copy below contents to .mobileconfig file and edit it

  4. Install the Profile

    1. Generate the profile with your desired settings
    2. Click "Download Profile" to save the .mobileconfig file
    3. Transfer the file to your iOS device (via AirDrop, email, etc.)
    4. On your iOS device, open the file and follow the installation prompts
    5. Go to Settings → General → VPN & Network → DNS to verify your DNS settings

    Compatible with:

    iOS 14 and later, iPadOS 14 and later
